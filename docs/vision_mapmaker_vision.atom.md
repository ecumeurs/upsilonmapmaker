# New Atom

## INTENT
Define the vision for UpsilonMapMaker as the procedural architect of the tactical battlefield.

## THE RULE / LOGIC
- **Core Role:** Generate diverse, fair, and tactically interesting game boards using procedural generation algorithms.
- **Goals:**
  - **Procedural Variety:** Ensure no two battles are the same by varying terrain, height, and obstacles.
  - **Winnability:** Guarantee that all generated maps are pathable and balanced for both teams.
  - **Performance:** Generate complex boards in milliseconds to support instant matchmaking.
